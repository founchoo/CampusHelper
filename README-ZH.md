# 校园助手
</div>

## 简介

欢迎来到校园助手的 Github 开源页面, 这是一款专为**湖北工业大学**在校学生服务的第三方教务管理系统，运行于**安卓**平台。

## 功能一览

目前，该应用提供如下功能：

- 查看课表
  - 切换到其他周查看其他周的课程
- 成绩查询
  - 排名查询（包括年级、专业、班级排名）
  - GPA 及算术平均分的自动计算
  - 成绩筛选（学年、课程类型）
